<h2>Password Toggle</h2>

A simple and user-friendly password visibility toggle built using vanilla JavaScript. This project enhances form usability by allowing users to show or hide their password input with a single click.

🚀 Features
- Toggle password visibility (show/hide)
- Instant UI update on user interaction
- Clean and minimal interface
- Responsive design

🛠️ Tech Stack
1. HTML5
2. CSS3
3. JavaScript (Vanilla JS)


🧠 What I Learned?
- DOM manipulation using JavaScript
- Handling user events (click interactions)
- Updating UI dynamically

📂 Project Structure
password-toggle/
│── index.html
│── style.css
│── script.js
│── README.md

🧩 How It Works
The password input field switches between type="password" and type="text"
A toggle button/icon listens for click events
JavaScript updates both the input type and the icon dynamically
